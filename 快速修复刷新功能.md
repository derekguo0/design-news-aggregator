## 🚀 快速修复（只需5分钟）

### 第一步：获取Vercel Token

1. 打开 https://vercel.com
2. 右上角头像 → **Settings**
3. 左侧 **Tokens** → **Create Token**
4. 命名为 `GitHub Actions Deploy`
5. **复制Token**（只显示一次！）

### 第二步：获取项目ID

在项目目录运行：
```bash
cd /Users/ithppc02110/Documents/推送机器人
npx vercel link
cat .vercel/project.json
```

会看到：
```json
{
  "orgId": "team_xxxx",
  "projectId": "prj_xxxx"
}
```

**或者**在Vercel控制台：
- 项目 → Settings → General → 找到 Project ID

### 第三步：添加GitHub Secrets

1. 打开 https://github.com/[你的用户名]/[仓库名]/settings/secrets/actions
2. 点击 **New repository secret** 添加以下3个：

```
Name: VERCEL_TOKEN
Value: [第一步的Token]

Name: ORG_ID  
Value: [第二步的orgId]

Name: PROJECT_ID
Value: [第二步的projectId]
```

### 第四步：测试

1. 访问 https://github.com/[你的仓库]/actions
2. 点击 **Run workflow** → **Run workflow**
3. 等待运行完成（应该显示绿色✅）
