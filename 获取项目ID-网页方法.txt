🌐 从Vercel控制台获取项目ID（最简单！）
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

⭐ 推荐使用这个方法！无需命令行操作！

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

第一步：获取 Project ID
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

1. 打开浏览器，访问：https://vercel.com

2. 登录您的账户

3. 在主页找到您的项目（可能叫 design-news-aggregator 或类似名称）

4. 点击项目名称进入

5. 点击顶部的 "Settings" 标签

6. 左侧菜单选择 "General"

7. 向下滚动，找到 "Project ID" 部分

8. 复制显示的ID（格式：prj_xxxxxxxxxxxxxxxxxxxx）

✏️ Project ID: ___________________________________

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

第二步：获取 Organization ID
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

方法1 - 从项目URL获取（最简单）：

查看浏览器地址栏的URL：
https://vercel.com/[这里就是orgId]/[项目名]

例如：
https://vercel.com/your-username/design-news
                   ^^^^^^^^^^^^^ 这就是 orgId

✏️ Organization ID: ___________________________________

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

方法2 - 从账户设置获取：

1. 点击右上角头像 → Settings

2. 如果是个人账户：
   - 在 General 标签查看 User ID

3. 如果是团队账户：
   - 左侧选择团队名称
   - 在 General 标签查看 Team ID

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

方法3 - 使用浏览器开发者工具：

1. 在项目页面按 F12 打开开发者工具

2. 切换到 Network 标签

3. 刷新页面

4. 查找请求URL中包含的 teamId 或 orgId 参数

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

信息汇总：
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

获取完成后，记录这两个值：

PROJECT_ID:  prj_________________________
ORG_ID:      ____________________________
             (可能是 team_xxx 或直接是用户名)

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

完成后的下一步：
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

将这两个ID添加到GitHub Secrets：

访问：https://github.com/[你的用户名]/[仓库名]/settings/secrets/actions

添加：
- Name: PROJECT_ID, Value: [您的项目ID]
- Name: ORG_ID, Value: [您的组织ID]

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

💡 提示：这个方法比命令行更简单，不需要登录CLI！
