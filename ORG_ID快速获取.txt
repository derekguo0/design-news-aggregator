🎯 ORG_ID 在哪找？（10秒钟搞定！）
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

最简单的方法：看浏览器地址栏！⭐⭐⭐

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

第1步：打开 Vercel 网站
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

在浏览器输入：https://vercel.com
登录您的账户

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

第2步：进入您的项目
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

在主页面找到您的项目，点击进入
（可能叫 design-news-aggregator 或类似名称）

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

第3步：看浏览器地址栏
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

地址栏显示的URL格式：

┌──────────────────────────────────────────────────────┐
│ https://vercel.com/[ORG_ID]/[项目名称]              │
│                     ^^^^^^^                          │
│                     这就是！                         │
└──────────────────────────────────────────────────────┘

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

举例说明：
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

例子1：
URL: https://vercel.com/zhang-designer/my-project
                       ^^^^^^^^^^^^^^
ORG_ID = zhang-designer

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

例子2：
URL: https://vercel.com/my-team/design-news
                       ^^^^^^^
ORG_ID = my-team

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

例子3：
URL: https://vercel.com/john-doe-123/app
                       ^^^^^^^^^^^^
ORG_ID = john-doe-123

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

第4步：复制 ORG_ID
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

从地址栏复制 vercel.com/ 后面、项目名前面的那段

✏️ 我的 ORG_ID: _______________________________

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

备用方法：从项目设置找
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

如果上面的方法不行：

1. 在项目页面，点击 Settings
2. 左侧选择 General
3. 找到 "Owner" 或 "Team" 信息
4. 那个名称就是 ORG_ID

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

快速确认方法
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

把您找到的 ORG_ID 放到这个地址：
https://vercel.com/[您的ORG_ID]

在浏览器打开，能看到您的项目列表就对了！✅

注意事项
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

✅ ORG_ID 可能包含：
   - 字母
   - 数字
   - 短横线 (-)
   - 下划线 (_)

✅ ORG_ID 格式：
   - 个人账户：通常是用户名（如：john-doe）
   - 团队账户：可能是 team_xxx 或团队名称

❌ 注意：
   - 不要包含空格
   - 不要包含 https://vercel.com/
   - 只要中间那段
